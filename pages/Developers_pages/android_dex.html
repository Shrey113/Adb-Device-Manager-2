Execution Pipeline

01

Package Verification

ADB verifies the helper package existence & status.

adb shell cmd package resolve-activity com.example.android_clone
02

Virtual Display Injection

Spawns a secondary display surface managed by Scrcpy. It can change according to the selected mode.

scrcpy --new-display=1920x1080/280 -f --start-app=com.example.android_clone --no-audio --window-title=AndroidDex
03

Input Service Handshake

Enables the accessibility service for touch injection.

adb shell settings put secure enabled_accessibility_services ...GlobalTouchService
04

Event Loop & Audio

Establishes TCP socket for real-time control events.

TCP: 5555 Audio: H.265 Input: HID
