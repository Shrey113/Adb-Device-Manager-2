## How to Use

1. Ensure both devices are connected to the same network.
2. Enable "Access Windows Files" in the Windows application settings.
3. Open the companion app on Android and navigate to the file access section.
4. Choose between Read-Only or Full Control mode.

## Requirements

- Companion app installed on Android
- Same Wi-Fi/LAN network
- Permission granted on Windows host
