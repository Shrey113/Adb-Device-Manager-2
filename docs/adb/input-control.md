# ⌨️ Input Control

Full keyboard, mouse, and gesture control for your Android device.

<img src="../../images/section_1/Keyboard%20%26%20Mouse%20Control.gif" width="600">

## Features

- **Keyboard Input** — Type directly on Android from Windows keyboard
- **Mouse Control** — Click, scroll, and drag with your mouse
- **Touch Gestures** — Swipe, pinch, and multi-touch support
- **Navigation Keys** — Home, Back, Recent apps buttons
- **Media Keys** — Volume, play/pause control

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Alt + I` | Toggle Input Control |
| `Alt + H` | Home |
| `Alt + B` | Back |
| `Alt + R` | Recent Apps |
| `Alt + Enter` | Toggle Fullscreen |
