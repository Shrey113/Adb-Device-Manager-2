1. Introduction

ADB Device Manager is designed to operate fully offline. All communication happens directly between your Windows PC and Android device using USB or local Wi-Fi.

  • No personal data is uploaded
  • No data transmitted to servers
  • No data stored externally
  • No data shared with third parties

The app processes data only on your devices.

5. Permissions Explanation

Permission Purpose
Notification Access Mirror notifications to Windows
READ_SMS / SEND_SMS Display and send SMS messages
READ_CONTACTS Show caller names
READ_CALL_LOG / PHONE_STATE Detect calls and show history
RECORD_AUDIO Audio during screen mirroring
READ/WRITE STORAGE File transfer between devices
INTERNET / NETWORK / WIFI Local Wi-Fi communication only
FOREGROUND SERVICE Keep connection stable

All permissions are used only for offline functionality. No permission is used to collect or transmit personal data.
